在node中的 Mongodb 以及 第三方封装的 mongoose 简单使用。

mongodb

MongoDB 是一个介于关系数据库和非关系数据库之间的产品，是非关系数据库当中功能最丰富，最像关系数据库的。

1. 安装相关

1.1 下载

官网下载地址 ：官网下载社区版

1.2 安装及环境配置

  • 安装

    下载的是 zip 解压到想要的路径即可

  • 配置

    我的电脑 –> 属性 –> 高级系统设置 –> 环境变量 –> 编辑用户变量的 path –> 添加解压mongondb路径下的bin目录即可

1.3 检测

在cmd控制台输入 mongod --version 出现版本信息及安装成功
